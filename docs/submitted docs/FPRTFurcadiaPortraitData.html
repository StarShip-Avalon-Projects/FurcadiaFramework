FPRT - Furcadia Portrait Data

FPRT format is normally used to enveope FSH data and present it specifically as a Furcadia portrait. It is normally a format you receive portraits in from the file server.

General Format

OFFSET

SIZE (bytes)

DESCRIPTION

0

6

“FPRT01” - FPRT magic number

6

1

Portrait width (90 / 0x5F)

7

1

Portrait height (90 / 0x5F)

8

1

Remappable? (1. Remappable, 0. Not remappable)

9

W*H

Data

NOTE: When receiving a portrait from the file server, the data is stored using XOR255+LZW algorithm! Data inside the portrait cache may likely be already decoded.
