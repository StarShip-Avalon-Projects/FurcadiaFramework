CHP - Portrait Cache Index File

CHP is an obsolete format, previously used by the portrait cache system to index data in the CHC files. Knowing where each portrait ID is in the sequence can help locate the offset of its data in the CHC file

General Format

OFFSET

SIZE (bytes)

DESCRIPTION

0

5

“DSB10” - DSB magic number

4

N*4

A list of portrait IDs, 4 bytes each
