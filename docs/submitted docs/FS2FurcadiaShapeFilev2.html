FS2 - Furcadia Shape File v2

FS2 files are a more advanced version of the FSH file with the following features:

  1. It actually has a magic number at the beginning (not much of an advancement)
  2. It allows encryption of the image data (although the size has to remain identical)
  3. With the “replacement shape” property, it may contain only the relevant shapes and still retain shape order by specifying which shape in the defaults does the new one replace. This allows for a significant optimization in file size and bandwidth required to transfer it.
