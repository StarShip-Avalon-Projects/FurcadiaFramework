FBJ - Furcadia Object Properties File

FBJ files store properties of every shape within the FSH/FS2 file that has the same name as the FBJ file in question. These files come separate off the FSH/FS2 files since it is easier for the Furcadia servers to parse this information and ignore an otherwise graphics-filled and unnecessary shape file.

General Format

OFFSET

SIZE (bytes)

DESCRIPTION

0

4

“FO01” - FBJ magic number

4

4

Amount of shape properties stored (do not exceed 16bit)

8

Shape property chunks

Property Chunk

Every property/shape chunk within the file has a structure mentioned below. The offset depends on where the chunk starts.

OFFSET

SIZE (bytes)

DESCRIPTION

+0

1

Flags (see below)

1

1|0

[DYNAMIC] X/Y Offset (either X or Y or none, depending on X/Y flags)

2

1|0

[DYNAMIC] Y Offset (optional, depending on X/Y flags)

Flags

0001 1111

   Y XSGW - WALKABLE flag    (0x01)

   | ||`--- GETTABLE flag    (0x02)

   | |`---- SITTABLE flag    (0x04)

   | `----- X offset present (0x08)

   `------- Y offset present (0x10)

The offset bytes are dynamic and their meaning highly depends on both the X and Y offset flags:

X = 1

X = 0

Y = 1

Byte1 = X offset, Byte2 = Y offset

Byte1 = Y offset, Byte2 = (doesn’t exist)

Y = 0

Byte1 = X offset, Byte2 = (doesn’t exist)

Byte1 and Byte2 don’t exist
