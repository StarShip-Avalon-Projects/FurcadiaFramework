FSH - Furcadia Shape File

FSH files were initially used to store graphics/images of each graphical item (object/floor/wall/avatar/UI element/etc.) within Furcadia. It is still used to this day, though a more advanced FOX format supersedes it and should be used wherever possible.

General Format

OFFSET

SIZE (bytes)

DESCRIPTION

0

2

Amount of shapes within the file

2

nShapes*2

Chunk size of each shape from 0 to n-1

2+nShapes*2

Image Chunk

File Tail
