TXB - DragonSpeak Text File

TXB files are generated off of DS files and contain just the text strings used in DragonSpeak. The strings are indexed and the index is later used in the binary DSB files.

General Format

The structure of a TXB file is as follows:

DST10\n

0:Master Socket kidnapped my kiwi!\n

1:String 1\n

2:String 2\n

...

#:String #\n

Elements

DST10

TXB file magic number

0:Master Socket...

Default fallback string (normally used with unknown indices)

#:String...

DragonSpeak string and its index as referenced by DSB files
